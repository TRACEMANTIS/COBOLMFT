# PERFORM VARYING

COBOL has a single loop verb that does a lot. The most common form is
`PERFORM ... VARYING ... UNTIL`:

```cobol
PERFORM VARYING I FROM 1 BY 1 UNTIL I > 10
    ADD I TO TOTAL
END-PERFORM.
```

Numeric variables need a PIC (picture) clause that declares both a type
and a width. `PIC 9(4)` is "four numeric digits, unsigned" — the rough
COBOL equivalent of a 4-digit unsigned decimal.

Your task: write a program that sums the integers 1 through 10 and
prints `SUM=55`. Use `DISPLAY` with a numeric edited PIC for clean
output, or just `DISPLAY TOTAL` if you don't mind leading zeroes (the
expected output regex accepts either).
